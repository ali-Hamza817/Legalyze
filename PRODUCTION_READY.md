## 🚀 How Reviewers Can Run the Product

### 3-Step Quick Start

**Step 1: Clone & Configure** (1 minute)
```bash
git clone https://github.com/ali-Hamza817/Legalyze.git
cd Legalyze
cp .env.example .env
# Edit .env with your API keys
```

**Step 2: Run with Docker** (1 minute)
```bash
docker-compose up --build
```

**Step 3: Access** (immediate)
- **Frontend**: http://localhost:3000
- **API**: http://localhost:5000
- **Test User**: admin@legalyze.com / Admin@123

### Total Time: ~3-5 minutes (depending on internet speed)
